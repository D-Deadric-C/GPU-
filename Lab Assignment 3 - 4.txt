Q1. Consider a scenario using OpenMP, you have to parallelize a program that performs matrix
multiplication. Your task is to implement parallelization using both static and dynamic
scheduling, and compare the execution time of each approach.
Using C/C++; OR Python Programming Environments
For input values, you have to consider dataset files of 10000 X 10000 data values.
Implement a serial version of matrix multiplication using C/C++; OR Python
Parallelize the matrix multiplication using OpenMP with static scheduling
Parallelize the matrix multiplication using OpenMP with dynamic scheduling.
Measure the execution time of each parallelized version for various matrix sizes.
Compare the execution times and discuss the advantages and disadvantages of static and
dynamic scheduling in this context.
Q2. Write a OpenMP program which should print the series of 2 and 4. Make sure both should be
executed by different threads. Compare the execution times.
Q3. Consider a scenario where you have a shared variable total_sum that needs to be updated
concurrently by multiple threads in a parallel program. Implement a simple parallel program
that initializes an array of integers and calculates the sum of its elements concurrently using
OpenMP.
Identify potential issues with concurrent updates to the total_sum variable in the parallelized
version of the program.
Modify the program to use OpenMP's critical/atomic directive to ensure synchronized access
to the total_sum variable.
Measure and compare the performance of synchronized versions against the unsynchronized
implementation. Compare the execution time of each approach.
